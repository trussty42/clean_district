# Чистый квартал
Интеллектуальная платформа для утилизации и переработки отходов и вторсырья.

## О проекте
**«Чистый квартал»** — централизованная веб-платформа, систематизирующая и упрощающая процесс раздельного сбора, сортировки и сдачи вторсырья. Платформа разработана в рамках учебного проекта УрФУ и направлена на создание единой экосистемы граждан, желающих правильно утилизировать отходы, и приёмные пункты, стремящиеся привлечь сырьё и клиентов, обеспечив всех участников исчерпывающей и актуальной информацией.

## Технологический стек
- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Python 3.13, Django 6.0, Django REST Framework, PostGIS 17-3, Docker
- **Архитектура:** модульная структура, MPA (Multi-Page Application), проект развернут на сервере
- **Инструменты:** Git, VS Code, Figma, Chrome DevTools, Postman, Swagger Editor, ReDoc, Docker Desktop

## Реализованный функционал

### Главная страница
- Блок «О нас» с информацией о платформе
- Пошаговая инструкция о том, как правильно сдавать вторсырьё
- Лента новостей от организаций
- Быстрый доступ к карте пунктов и каталогу отходов

### Карта пунктов приёма
- Интерактивная карта с метками приёмных пунктов
- Дублирующий список пунктов
- Фильтры по типу принимаемых отходов, удалённости, рейтингу и времени работы
- Поиск по адресу или названию пункта
- Карточка пункта: график работы, контакты, принимаемые материалы, актуальные цены, отзывы

### Каталог отходов
- Фильтрация по типу отхода
- Поиск по названию отхода
- Карточки отходов со средней ценой за кг
- Детальная страница отхода: правила подготовки (помыть, сжать, снять крышку/этикетку), что нельзя сдавать и во что будет переработано
- Интеграция с формой быстрой сдачи: подбор пункта с лучшей ценой на выбранный тип

### Личный кабинет пользователя
- Регистрация и авторизация
- Профиль с редактированием данных и загрузкой аватара
- История сдач с возможностью оставить отзыв
- Система из 12 достижений с прогресс-трекингом
- Модальное окно для быстрой регистрации новой сдачи
- Интерактивные подсказки по подготовке вторсырья перед визитом

### Кабинет организации (Приёмные пункты)
- Личный кабинет с редактированием данных
- Управление филиалами
- Динамический блок цен по типам отходов
- Вкладка «Аналитика»: просмотры, объёмы сдач, средний рейтинг, распределение оценок
- Управление отзывами: просмотр, ответы пользователям
- Публикация новостей и акций
- Управление сотрудниками
